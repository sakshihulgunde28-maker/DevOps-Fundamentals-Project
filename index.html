AWS EC2 Issues

Problem: SSH connection failed
Solution: Check Security Group rules, key permissions, and instance status.

Linux Issues

Problem: Permission denied error
Solution: Check file ownership and use chmod/chown commands.

Apache2 Issues

Problem: Website not loading
Solution: Check Apache service status and port 80 configuration.

Git & GitHub Issues

Problem: Git push or pull failure
Solution: Verify repository URL, branch name, and authentication.

Jenkins Issues

Problem: Pipeline build failed
