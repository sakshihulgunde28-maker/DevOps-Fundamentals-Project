Project Objective

The objective of this project is to learn the fundamentals of DevOps by deploying a website on AWS EC2. This project includes Linux administration, Apache web server configuration, Git version control, and GitHub integration.

Learning Outcomes

  • Understanding AWS EC2
  • Learning Linux commands
  • Installing and configuring Apache
  • Using Git and GitHub
  • Managing project files
  • Understanding DevOps workflow
