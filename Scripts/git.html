AWS Setup

EC2 Instance Creation

An Ubuntu EC2 instance was created on AWS cloud platform to host the website.

Security Groups

Security Groups were configured to allow HTTP traffic on Port 80 and SSH traffic on Port 22.

Apache Installation

    sudo apt update
    sudo apt install apache2 -y
    sudo systemctl start apache2
    sudo systemctl enable apache2

Apache web server was installed and configured to host the project website.
